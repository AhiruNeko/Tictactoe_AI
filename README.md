# 🎮 Tictactoe_AI - 井字棋人工智能

一个基于深度强化学习的井字棋AI，通过自弈学习掌握最优策略，能与人类玩家进行对抗。

## 📋 项目概述

本项目实现了一个会下井字棋的人工智能系统。AI采用**策略网络（Policy Network）** + **价值网络（Value Network）** + **蒙特卡洛树搜索（MCTS）** 的组合方案，能够通过自我对弈自主学习，无需人工标注的训练数据。

## 🔧 技术栈

- **框架**: TensorFlow & Keras
- **算法**: 深度强化学习 + 蒙特卡洛树搜索
- **语言**: Python 100%

## 🧠 核心算法

### 网络架构

#### 策略网络 (Policy Network)
- **输入**: 当前棋盘状态
- **输出**: 9个概率值（9个棋盘位置的落子概率）
- **激活函数**: Sigmoid（输出值在0-1之间）

#### 价值网络 (Value Network)  
- **输入**: 当前棋盘状态
- **输出**: 单个评估值（当前局面的价值判断）
- **激活函数**: Sigmoid（输出值在0-1之间）

### 训练算法

训练过程采用**策略网络、价值网络和蒙特卡洛树搜索（MCTS）** 的结合：

1. **MCTS节点扩展**: 采用 **UTC（Upper Confidence bound applied to Trees）算法**，探索系数设置为 **1.4**
2. **模拟与回溯**: MCTS经过多次模拟对弈后进行回溯，更新节点数据
3. **网络优化**: 用MCTS的结果训练策略网络和价值网络
4. **双向反馈**: 训练后的策略网络和价值网络反过来指导MCTS的进行，形成正反馈循环

### 对弈策略

在与玩家对弈时，采用以下决策方法：

1. **网络与搜索结合**: 综合策略网络和MCTS的输出
2. **加权决策**: 由于硬件限制，策略网络相对较弱，因此增加MCTS在决策中的权重
3. **值映射优化**: 将MCTS的输出通过特定函数进行映射，增大极端值的差异，防止误判
4. **最优决策**: 综合两个信息源的结果，选择评估值最大的位置落子

```

├── README.md                 # 项目��明文档
```

## 🚀 使用方法

### 环境要求

```bash
# 安装依赖
pip install tensorflow keras numpy
```

### 训练AI

```bash
# 基础训练
python train.py

# 特殊训练模式
python special_train.py
```

### 与AI对弈

```bash
python UI.py
```

或直接使用预测模块：

```bash
python predict.py
```

## ⚙️ 配置说明

- **MCTS探索系数 (UCT)**: 1.4
- **网络激活函数**: Sigmoid
- **输出范围**: [0, 1]
